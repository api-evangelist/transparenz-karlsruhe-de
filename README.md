# Transparency Portal for the City of Karlsruhe, Germany (transparenz-karlsruhe-de)

Transparency Portal for the City of Karlsruhe, Germany is a open data portal open-data portal for Germany running **CKAN**, exposing a standardized catalog API over ~150 datasets.
